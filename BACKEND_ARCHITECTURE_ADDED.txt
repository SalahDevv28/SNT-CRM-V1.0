✓ DATA SECURITY LAYERS (4 layers)

  Layer 1: Row Level Security (RLS)
  ├─ Enforced at database level
  ├─ Applies to all queries
  └─ Cannot be bypassed

  Layer 2: Authentication
  ├─ JWT tokens from GoTrue
  ├─ Token expiration (15min/7-day)
  └─ OAuth providers

  Layer 3: Role-Based Access Control (RBAC)
  ├─ Admin (full access)
  ├─ Broker/Manager (team access)
  ├─ Agent (assigned leads only)
  └─ Client (read-only)

  Layer 4: Data Validation
  ├─ Database constraints
  ├─ PostgreSQL type enforcement
  └─ Trigger-based validation

✓ LEAD SCORING & ROUTING LOGIC

  Automated Qualification System

  ├─ Meetings: +20 per meeting (max 40)
